## 📂 Featured Projects

### 🎯 osu! Automation
Experimental automation prototype written in **C#** that parses `.osu` beatmap files and replays hit circles using Win32 input simulation.

Key topics:
- beatmap parsing
- coordinate mapping
- timing synchronization
- Windows input APIs

---

### 🧠 Parallel Tic-Tac-Toe AI
Implementation of a configurable **Tic-Tac-Toe AI** using **Minimax with Alpha-Beta pruning** written in **C**.

Features:
- variable board size
- optional **OpenMP parallelization**
- benchmarking and performance plots

---

### 🖥 8086 Microsystem
Academic project implementing a **custom microsystem around the Intel 8086**.

Includes:
- memory mapping
- peripheral decoding
- assembly routines
- EasyEDA schematics

---

### 🗺 Unix Treasure Hunt System
Filesystem-based treasure hunt manager written in **C for Unix/Linux**.

Concepts used:
- process management
- pipes
- signals
- binary file I/O
- directory traversal
